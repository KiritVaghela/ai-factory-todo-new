# Todo App## DescriptionA simple and efficient Todo application to manage tasks.## Features- **User Authentication**: Allow users to sign up, login, and logout.- **CRUD Operations for Todos**: Create, Read, Update, and Delete tasks.- **Filter and Sort Todos**: Users can filter tasks by status and sort them.- **Due Date and Reminders**: Set due dates for tasks and send reminders.- **Responsive Design**: Ensure the app works well on both desktop and mobile.## Technology Stack- **Frontend**: React, Redux, Bootstrap- **Backend**: Node.js, Express, MongoDB- **DevOps**: AWS, Docker, GitHub Actions## Installation1. Clone the repository: `git clone <repository-url>`2. Navigate to the project directory: `cd todo-app`3. Install dependencies: `npm install`## Running the Application- To run the backend server: `node server.js`- To run using Docker: `docker-compose up`
