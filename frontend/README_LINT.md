# Linting and Static Analysis for Frontend

## How to Run

1. Ensure dependencies are installed:

   ```sh
   npm install eslint eslint-plugin-react eslint-plugin-import --save-dev
   ```

2. To run linter and static analysis on all JS/JSX files:

   ```sh
   node lint-and-analyze.js
   ```

This will check for syntax errors and import errors in all JavaScript and JSX files in the frontend.
